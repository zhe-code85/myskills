# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Purpose

Personal skills repository for collecting, refining, and evolving reusable agent skills. Currently tested with Claude Code and Codex; other platforms are untested.

## Repository Structure

Each skill is a self-contained top-level directory with `SKILL.md` as the entry point.

## Conventions

- README.md maintains a skill registry (name, status, description, path, dependency versions)
- When adding a new skill: create a top-level directory with `SKILL.md`, then add a row to the README.md skill table with its name, status, description, path, and dependency versions
